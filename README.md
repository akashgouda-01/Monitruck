# Monitruck – Fleet Monitoring System

A full-stack fleet monitoring dashboard for fleet managers.

## Features
- 🚛 Live GPS tracking
- 😴 Drowsiness detection
- ⛽ Fuel monitoring
- 🔔 Real-time alerts
- 🔐 Login for fleet managers

## Tech Stack
- React + Vite (Frontend)
- Express + MongoDB (Backend)
- Socket.IO (Real-time)
- Shadcn UI + Tailwind CSS
